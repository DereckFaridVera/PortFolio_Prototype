Sistema de Gestión Hospitalaria

Desarrollo del Proyecto

Desarrollado con Java Spring Boot para el backend y Angular para el frontend, con base de datos PostgreSQL y cumplimiento HIPAA.

Indicaciones

  • Gestión de historias clínicas electrónicas
  • Programación de citas y turnos
  • Facturación y gestión de seguros
  • Alertas médicas y seguimiento de pacientes

Objetivo del Proyecto

Crear un sistema integral que optimice los procesos administrativos y clínicos en instituciones de salud, mejorando la calidad de la atención al paciente.

Conclusiones

El sistema redujo en un 40% el tiempo de espera de los pacientes y mejoró la precisión en la gestión de medicamentos. La implementación cumplió con todos los requisitos de privacidad y seguridad médica.
