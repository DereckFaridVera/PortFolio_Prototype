Sistema de Monitoreo IoT

Desarrollo del Proyecto

Implementado con Arduino para los dispositivos, Node.js para el servidor y React para la interfaz de monitoreo en tiempo real.

Indicaciones

  • Monitoreo de sensores en tiempo real
  • Alertas automáticas por umbrales
  • Historial de datos y tendencias
  • Control remoto de dispositivos

Objetivo del Proyecto

Desarrollar una solución IoT completa para el monitoreo remoto de parámetros ambientales y operativos en entornos industriales y domésticos.

Conclusiones

El sistema demostró ser altamente confiable en la transmisión de datos en tiempo real y permitió la detección temprana de anomalías. La arquitectura escalable facilitó la adición de nuevos dispositivos y sensores.
