Aplicación Móvil de Productividad

Desarrollo del Proyecto

Desarrollada con React Native para compatibilidad multiplataforma, utilizando Redux para la gestión del estado y Firebase para la sincronización en tiempo real.

Indicaciones

  • Creación y gestión de tareas con fechas límite
  • Sistema de hábitos con seguimiento diario
  • Notificaciones push personalizadas
  • Sincronización automática entre dispositivos

Objetivo del Proyecto

Desarrollar una herramienta accesible que ayude a los usuarios a mejorar su productividad personal mediante la organización efectiva de tareas y el seguimiento de hábitos positivos.

Conclusiones

La aplicación logró un alto nivel de satisfacción entre los usuarios beta, con un 85% reportando mejoras significativas en su organización personal. La arquitectura elegida permitió una excelente experiencia de usuario en ambas plataformas.
