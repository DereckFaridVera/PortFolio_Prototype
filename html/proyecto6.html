Aplicación de Reconocimiento Facial

Desarrollo del Proyecto

Implementado con Python, OpenCV y TensorFlow, con una API REST para integración con aplicaciones web y móviles.

Indicaciones

  • Autenticación biométrica segura
  • Detección de rostros en tiempo real
  • Almacenamiento seguro de templates faciales
  • Integración con sistemas de control de acceso

Objetivo del Proyecto

Desarrollar un sistema de autenticación biométrica confiable y seguro que pueda reemplazar métodos tradicionales como contraseñas y tarjetas de acceso.

Conclusiones

El sistema alcanzó una precisión del 98.5% en condiciones controladas y demostró ser resistente a intentos de spoofing. La implementación respetó estándares de privacidad y protección de datos biométricos.
